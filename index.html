ParaVM

Parallella Virtual Machine

Git       Wiki       Issues       Jenkins       IRC

ParaVM is a virtual machine written specifically for the Epiphany processor architecture. More specifically, it is meant to run on a 'regular' host CPU (such as x86 or ARM) where it compiles code (in a JIT fashion) to Epiphany machine code that is executed on Epiphany accelerator cores in an isolated way similar to the concurreny model employed by Erlang. One such system is the Parallella board which ParaVM is primarily written for.

ParaVM is not meant to be usable by itself; rather, it's a platform that programming languages can compile to in order to take advantage of the Epiphany architecture. The project is open source under the MIT License and we encourage everyone to get involved with the design and development of the virtual machine. Please see the contribution guidelines if you are interested in contributing.
